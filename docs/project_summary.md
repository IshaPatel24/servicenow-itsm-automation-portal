# ITSM Automation Portfolio Project

## Project Overview

This project demonstrates an end-to-end IT Service Management (ITSM) automation solution built on ServiceNow. The implementation includes Service Catalog request management, approval workflows, catalog task automation, and Jira integration using REST APIs.

The solution simulates a real-world enterprise service fulfilment process where user requests are automatically processed through approval, task creation, fulfillment, and closure stages.

---

## Objectives

- Automate Service Catalogue request fulfilment
- Manage Requests (REQ), Requested Items (RITM), and Catalogue Tasks
- Implement approval workflows
- Automate task closure and request completion
- Integrate ServiceNow with Jira using REST APIs
- Demonstrate ITSM best practices
